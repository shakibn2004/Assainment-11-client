## 🌟 Overview

**SparkLift** is a modern, premium crowdfunding platform designed to connect visionary creators with passionate supporters. Featuring a state-of-the-art luxury design language, it offers a seamless, beautiful, and highly interactive user experience across all devices. 

Whether you're an ambitious creator launching a dream project, a backer exploring the next big thing, or an admin overseeing the platform, SparkLift provides a dedicated, feature-rich interface tailored to your role.

## ✨ Key Features

- **🎨 Luxury Aesthetic:** A visually stunning interface utilizing glassmorphism, dynamic gradients, and fluid micro-animations.
- **🌗 Seamless Theming:** Full support for both Light and Dark modes using modern semantic CSS variables (`oklch`).
- **📱 Fully Responsive:** Carefully crafted layouts that scale flawlessly from large desktop monitors down to mobile devices, featuring native-like slide-out navigation drawers.
- **🎭 Role-Based Architecture:** Three distinct, fully mocked user experiences:
  - **Supporter:** Browse campaigns, track backed projects, manage rewards, and handle secure checkouts.
  - **Creator:** Create campaigns, monitor real-time funding progress, track page views, and manage backers.
  - **Admin:** View platform-wide statistics, revenue charts, user management, and campaign approval queues.
- **💳 Optimized Checkout:** A frictionless mock checkout flow with shipping address collection and secure payment gateway redirection.
- **📊 Interactive Analytics:** Beautiful, responsive charts built with Recharts to visualize funding metrics.

## 🛠️ Technology Stack

SparkLift is built using the most modern and robust tools in the React ecosystem:

### Frontend
- **[Next.js 14+](https://nextjs.org/)** - React framework using the App Router for unparalleled performance and SEO.
- **[React 18](https://react.dev/)** - The library for web and native user interfaces.
- **[TypeScript](https://www.typescriptlang.org/)** - For robust type-safe code and exceptional developer experience.

### Styling & UI
- **[Tailwind CSS](https://tailwindcss.com/)** - Utility-first CSS framework for rapid, custom UI development.
- **[Shadcn UI](https://ui.shadcn.com/)** - Beautifully designed, accessible, and customizable React components.
- **[Framer Motion](https://www.framer.com/motion/)** - Production-ready animation library used for parallax heroes and bento grid reveals.
- **[Lucide React](https://lucide.dev/)** - Beautiful & consistent icon toolkit.

### State & Data
- **[Zustand](https://docs.pmnd.rs/zustand/)** - A small, fast, and scalable bearbones state-management solution used for the global mock authentication state.
- **[Recharts](https://recharts.org/)** - A composable charting library built on React components.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v18.17.0 or higher)
- npm or yarn or pnpm

### Installation

1. **Clone the repository** (if applicable)
   ```bash
   git clone https://github.com/yourusername/fundbridge.git
   cd fundbridge
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Explore the app**
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🎮 Using the Demo

Since the platform currently uses mock data and state for demonstration purposes:
- **Authentication:** Click "Log In" or "Get Started" to instantly mock-authenticate.
- **Role Switching:** Once logged in, click your Avatar in the top right (or open the mobile menu) to switch between **Supporter**, **Creator**, and **Admin** roles to explore the different dashboard interfaces.
- **Checkout:** Navigate to any campaign, click "Back this project", and proceed through the checkout form to see the validation and routing logic.
